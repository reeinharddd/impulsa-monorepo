# @impulsa/ui

This library contains the shared UI components and design system for Impulsa applications.

## Purpose

To ensure visual consistency across all Impulsa applications (Web, Admin, etc.) while allowing for theme variations.

## Structure

- `src/lib/components`: Reusable components (Buttons, Inputs, Cards)
- `src/lib/theme`: Tailwind configuration and theme definitions
- `src/lib/layouts`: Shared layouts

## Usage

Import components from `@impulsa/ui` in your Angular applications.
